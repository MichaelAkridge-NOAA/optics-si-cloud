💻 VS Code Server Setup

Browser-based VS Code on Cloud Workstations (port 8443)

Overview

code-server runs VS Code in your browser. It's useful as:

  • An alternative when the Cloud Workstation IDE is slow
  • A way to access VS Code from any device with a browser
  • A lightweight option for quick edits

Port: 8443

Time: ~3 minutes

Step 1: Run the Setup Script

curl -sL https://raw.githubusercontent.com/MichaelAkridge-NOAA/optics-si-cloud-tools/main/scripts/setup_code_server.sh | bash

The script downloads the latest code-server, configures it, and starts it automatically.

Step 2: Access code-server

code-server runs on port 8443. Access it via Cloud Workstation port forwarding:

  1. In the Cloud Workstations console, find your workstation
  2. Look for port 8443 in the port forwarding section
  3. Click the forwarding URL

Or construct the URL manually:

https://8443-YOUR-WORKSTATION.cloudworkstations.dev
No password required — Cloud Workstation IAM handles authentication.

Step 3: Reload Shell (for commands)

source ~/.bashrc

Management Commands

Command Description
code-server-startStart code-server
code-server-stopStop code-server
code-server-statusCheck if running
code-server-logsView logs

Installing Extensions

Install extensions from the command line:

code-server --install-extension ms-python.python
code-server --install-extension esbenp.prettier-vscode

Or use the Extensions panel in the browser UI.

Note: Some VS Code extensions may not work in code-server. Check the Open VSX Registry for compatible extensions.

Pre-installed Extensions

The setup script installs these extensions:

  • Python — Python language support
  • Prettier — Code formatter
  • ESLint — JavaScript linter
  • GitLens — Git supercharged

Configuration

code-server config is at:

~/.config/code-server/config.yaml

Default settings:

  • Bind: 0.0.0.0:8443
  • Auth: none (IAM handles it)
  • Telemetry: disabled

Persistence

✓ Everything persists across restarts!
  • code-server: ~/.code-server
  • Extensions: ~/.code-server/extensions
  • User data: ~/.code-server/data
  • Config: ~/.config/code-server
