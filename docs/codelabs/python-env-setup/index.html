🐍 Python Environment Setup

Persistent pyenv + virtual environments for Cloud Workstations

Overview

This guide installs pyenv for Python version management and creates persistent virtual environments that survive Cloud Workstation restarts.

What you'll get:

  • pyenv installed in ~/.pyenv (persists)
  • Python 3.12 (or your choice) installed
  • Default venv in ~/.venvs/default
  • Helper commands: venv-default, venv-create, venv-list

Time: ~5 minutes

Step 1: Run the Setup Script

Download and run the setup script:

curl -sL https://raw.githubusercontent.com/MichaelAkridge-NOAA/optics-si-cloud-tools/main/scripts/setup_python_env.sh | bash

Or to install a specific Python version:

curl -sL https://raw.githubusercontent.com/MichaelAkridge-NOAA/optics-si-cloud-tools/main/scripts/setup_python_env.sh | bash -s -- 3.11
Note: The script will install build dependencies (requires sudo) and compile Python from source. This takes a few minutes.

Step 2: Reload Your Shell

After installation, reload your shell to use the new commands:

source ~/.bashrc

Step 3: Activate the Default Environment

Activate the pre-created default virtual environment:

venv-default

You should see (default) in your prompt.

Verify Python:

python --version
which python

Using Virtual Environments

List Available Environments

venv-list

Create a New Environment

venv-create myproject

Then activate it:

source ~/.venvs/myproject/bin/activate

Deactivate

deactivate

Managing Python Versions with pyenv

List Installed Versions

pyenv versions

Install a New Version

pyenv install 3.11.8

Set Global Default

pyenv global 3.11.8

List Available Versions

pyenv install --list | grep "^\s*3\."

Pre-installed Packages

The default environment comes with these packages:

  • numpy - Numerical computing
  • pandas - Data manipulation
  • matplotlib - Plotting
  • scikit-learn - Machine learning
  • requests - HTTP client
  • ipython - Enhanced Python shell
  • black - Code formatter
  • ruff - Fast linter
  • pytest - Testing framework

Persistence

✓ Everything persists across restarts!
  • pyenv: ~/.pyenv
  • Virtual environments: ~/.venvs/
  • Shell config: ~/.bashrc
