🔐 Git & SSH Setup

Configure Git and SSH keys for GitHub, GitLab, and Cloud Source Repos

Overview

This guide configures:

  • Git identity (name, email)
  • Useful Git defaults and aliases
  • SSH key for secure authentication
  • SSH config for GitHub, GitLab, Cloud Source Repos

Time: ~2 minutes

Step 1: Run the Setup Script

curl -sL https://raw.githubusercontent.com/MichaelAkridge-NOAA/optics-si-cloud-tools/main/scripts/setup_git_ssh.sh | bash

Or with your name and email pre-filled:

bash setup_git_ssh.sh "Your Name" "your@email.com"

Step 2: Copy Your Public Key

The script outputs your public SSH key. Copy it:

cat ~/.ssh/id_ed25519.pub

Step 3: Add Key to Git Providers

GitHub

  1. Go to github.com/settings/ssh/new
  2. Paste your public key
  3. Give it a title (e.g., "Cloud Workstation")
  4. Click "Add SSH key"

GitLab

  1. Go to gitlab.com/-/user_settings/ssh_keys
  2. Paste your public key
  3. Click "Add key"

Google Cloud Source Repositories

  1. Go to source.cloud.google.com/user/ssh_keys
  2. Click "Register SSH Key"
  3. Paste your public key

Step 4: Test the Connection

# GitHub
ssh -T git@github.com

# GitLab
ssh -T git@gitlab.com

You should see a welcome message like:

Hi username! You've successfully authenticated...

Git Aliases

The script adds these handy aliases:

Alias Command Description
git ststatus -sbShort status
git cocheckoutCheckout
git brbranchBranch
git cicommitCommit
git lglog --oneline --graphPretty log
git lastlog -1 HEAD --statLast commit
git unstagereset HEAD --Unstage files

Cloning Repositories

Now you can clone using SSH:

# GitHub
git clone git@github.com:username/repo.git

# GitLab
git clone git@gitlab.com:username/repo.git

# Cloud Source Repos
git clone ssh://source.developers.google.com/p/PROJECT/r/REPO

Persistence

✓ Everything persists across restarts!
  • SSH keys: ~/.ssh/
  • Git config: ~/.gitconfig
